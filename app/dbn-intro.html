drawing and painting



paper

sets the background color of the dbn 'paper', by filling the dbn area with a color 

    paper 100  // make the paper black
    paper 0    // make the paper white
    paper 40   // make the paper 40% gray
pen

changes the pen color to be used for drawing commands, like 'line'. 

    pen 100  // pick up a 100% black pen
    pen 0    // pick up a white pen
line

draws a line between two points. the syntax is line left top right bottom. the lower-left corner of the drawing area is (0, 0) and the top-right is (100, 100). 

    // draw a line from the lower left to the upper right
    line 0 0 100 100  

    // draw a line from (20, 40) to (80, 90)
    line 20 40 80 90
